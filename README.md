# 枪械肢体命中判定系统

一个用于迷你世界 UGC 3.0 的 Lua 脚本，可在玩家开枪时精确判定击中目标（玩家/生物）的肢体部位：**头部、躯干、腿部、左臂、右臂**。采用高精度步进射线检测与角色碰撞箱 AABB 求交，支持动态获取目标，并提供了丰富的可调参数以适应不同模型。

## 特性

- **高精度射线检测**：步长可配置（默认 0.0075 米），射线与方块碰撞箱逐点检测，确保子弹不穿墙。
- **精确肢体判定**：基于击中点相对于角色脚底的高度比例和**横向偏移**（垂直于面朝方向）进行判断，避免身体前后侧误判。
- **自动排除自身**：射线检测时自动忽略开枪玩家，不会击中自己。
- **支持玩家与生物**：动态获取视野内所有玩家和生物（通过 `Area:GetAllCreaturesInAreaRange`），无需手动维护列表。
- **灵活配置**：所有判定阈值均可在 `CONFIG` 表中调整，适配不同体型的角色模型。
- **性能优化**：预先计算目标 AABB 交点并按距离排序，射线步进检测到障碍物立即停止，减少无用遍历。

## 配置参数

在脚本开头的 `CONFIG` 表中可调整以下参数：

| 参数名 | 类型 | 默认值 | 说明 |
| :-: | :-: | :-: | :-: |
| `EYE_HEIGHT` | number | 1.6 | 玩家眼睛离脚底的高度（米） |
| `MAX_DIST` | number | 100 | 最大射程（米） |
| `STEP_SIZE` | number | 0.0075 | 射线步长（米），越小精度越高但性能稍降 |
| `LEG_RATIO` | number | 0.2 | 腿部最大高度比例（相对于总身高） |
| `HEAD_RATIO` | number | 0.8 | 头部最小高度比例 |
| `ARM_START_RATIO` | number | 0.4 | 手臂区域起始高度比例 |
| `LATERAL_COEFF_HIGH` | number | 0.4 | 手臂区域横向偏移系数（×半宽） |
| `LATERAL_COEFF_LOW` | number | 0.5 | 低高度区域横向偏移系数（下垂手掌等） |
| `LATERAL_COEFF_EXTREME` | number | 0.7 | 极外侧横向偏移系数（任何高度） |

> **说明**：横向偏移指击中点到角色身体中线的左右距离（绝对值），单位为米。系数乘以角色的实际半宽（从 `GetBoundSzie` 获取）得到判定阈值。例如半宽 0.48 米，`LATERAL_COEFF_HIGH = 0.4` 意味着横向偏移大于 0.192 米即可判为手臂。

## 使用方法

1. **导入脚本**  
   将提供的 Lua 代码保存为 `.lua` 文件，并在 UGC 编辑器中添加到你的地图（挂载到世界组件或任意对象上）。

2. **事件监听**  
   脚本会自动监听 `TriggerEvent.PlayerGunAction`（玩家开枪事件），无需额外操作。

3. **测试**  
   进入游戏后，持枪射击玩家或生物，控制台会输出：

   ```text
   玩家 <迷你号> 开枪击中 <目标ID> 的 <部位> (击中点: x, y, z)
   ```

4. **调整参数**  
   根据你的模型实际测试结果，修改 `CONFIG` 中的阈值，直到满足判定需求。

## 注意事项

- **生物自动获取**：需要 `Area:GetAllCreaturesInAreaRange` 接口支持，该接口通常可用。若不存在，可替换为其他获取生物列表的方式（如手动维护生物数组）。
- **方块阻挡**：射线检测会识别非空气方块（`Block:GetBlockID ≠ 0`）并停止穿透，因此子弹无法穿过墙体。
- **性能提示**：步长 0.0075 米时，最大射程内约遍历 `13333` 次 `Block:GetBlockID`，加上目标数量通常很少，现代设备可流畅运行。若遇性能瓶颈，可将 `STEP_SIZE` 增大至 `0.01` ~ `0.02` 米。
- **朝向计算**：肢体左右判定基于角色的 `GetFaceYaw` 方向，确保朝向正确。如果左右臂总是相反，可将 `dot > 0` 改为 `dot < 0`。
